Knowledge hub plugin add knowledge-hub as custom post type to publish help/support articles in WordPress based site.

== Description ==

This plugin acts as a Stripe Checkout payment gateway add-on for the Islamic Donation plugin. During donation checkout, it redirects to Stripe Checkout to complete the payment and then redirects back to the main website's thank you page upon success. This add-on enables you to complete checkout using various payment methods available in your Stripe account. You can make both single and recurring donations with this payment method.

== Installation ==

1. Upload `knowledge-hub` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Config the addon plugin from Islamic Donation plugin payment settings
